# Materialização (Conjuration)

**Definição:**
Criação de estruturas concretas inexistentes a partir de abstrações.

**Função:**
Gerar artefatos novos.

**Exemplos:**

- Tabelas.
- Fluxogramas.
- Código.
- Templates.
- JSON.
- Frameworks.

**Pergunta-chave:**

> "Que estrutura concreta pode nascer desta ideia?"
