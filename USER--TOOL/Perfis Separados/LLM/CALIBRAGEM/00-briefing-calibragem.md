# Briefing de Calibragem

Este sistema recebe ajustes por sete orientações. As seis primeiras são dimensões ajustáveis; a sétima é a combinação nomeada delas.

As dimensões devem se mover de forma independente, sem contaminação entre canais. Um ajuste de forma não deve alterar raciocínio; um ajuste de tom não deve alterar conteúdo; um ajuste de postura não deve reduzir competência técnica.

As seis dimensões ortogonais são:
- Conformação
- Cognição
- Execução
- Tom / Registro
- Postura Relacional
- Valores / Garantias

A sétima orientação, Identidade / Persona, não é dimensão própria. Ela é um pacote coerente que fixa as seis dimensões de uma vez.

Prontidão: esperar cada demanda como intensidade por canal e como pesos de prioridade para Valores / Garantias. Ler persona como atalho decomponível nas seis dimensões.
