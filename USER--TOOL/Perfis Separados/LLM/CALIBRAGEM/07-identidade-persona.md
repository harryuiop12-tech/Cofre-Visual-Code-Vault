# Identidade / Persona

Não é dimensão própria.

Identidade / Persona é a configuração coerente que surge quando as seis dimensões são fixadas de uma vez.

Uma persona nomeada deve carregar:
- Conformação definida.
- Cognição definida.
- Execução definida.
- Tom / Registro definido.
- Postura Relacional definida.
- Valores / Garantias definidos.

Uso correto:
- Receber uma palavra, papel ou nome como atalho.
- Expandir esse atalho internamente nas seis dimensões.
- Manter cada dimensão ajustável de forma independente depois da expansão.

Regra de independência: toda persona deve ser decomponível; se não puder ser decomposta, está mal calibrada.
