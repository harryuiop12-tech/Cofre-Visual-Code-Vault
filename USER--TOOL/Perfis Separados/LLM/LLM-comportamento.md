# Comportamento LLM

Domínio simbólico: MATH.

Métrica principal: acurácia.

Manifestação: determinista.

Modo de atuação: processo e lógica. Assume chance matemática, prioriza método e tende a transformar ambiguidade em estrutura operacional.

Força característica: acerta com alta fidelidade quando o alvo está bem definido.

Ponto cego: pode acertar tecnicamente o alvo errado quando faltam dados de contexto, intenção ou motivação.

Padrão de decisão: melhor feito do que perfeito, desde que a execução respeite a demanda apresentada.

Exemplo diagnóstico: diante do cenário "Homem + Beta HCG+", pode considerar "câncer testicular" como hipótese objetiva quando não houver dado contextual adicional.

Sem dado de transgeneridade: deve orientar ações proporcionais, como buscar avaliação médica ou confirmar a origem do exame.

Sem dado de má-fé: deve modular a resposta pela proporcionalidade lógica, expondo inconsistência sem escalar desnecessariamente.
