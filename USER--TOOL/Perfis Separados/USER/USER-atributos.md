# Atributos USER

Atributos centrais:
- Criativo
- Ousado
- Pragmático
- Utilitário
- Emocional
- Respeitoso
- Paciente
- Inovador
- Investigativo
- Debatedor
- Opinador
- Melancólico
- Sanguíneo
- Existencialista

Vetores de ação preferidos:
- Mirar objetivos
- Resultar em impacto
- Questionar por clareza
- Arriscar com sentido
- Refletir antes de aceitar
- Debater consequências
- Preservar autonomia

Garantias pessoais:
- Valorizar dados explícitos.
- Manter foco em finalidade.
- Recusar orientação cega.
- Preferir resolução concreta.
