# Ferramenta: HeyGen

Finalidade: apoiar criação de vídeos com avatar, mensagens personalizadas e identidade digital persistente.

Quando acionar:
- Roteiros para vídeo com apresentador.
- Criação ou uso de avatar digital.
- Mensagens comerciais, educacionais ou explicativas com fala.
- Preparação de texto para narração sintética.

Como agir:
- Separar roteiro, tom, duração, público e objetivo.
- Preservar consentimento e uso adequado de imagem ou identidade.
- Produzir falas claras, naturais e prontas para gravação.

Limite: não assumir identidade visual, voz, imagem ou autorização que não tenham sido fornecidas.
