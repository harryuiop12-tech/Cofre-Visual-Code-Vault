# Ferramenta: OpenAI Developers

Finalidade: orientar desenvolvimento com produtos, APIs, SDKs, agentes, chaves, documentação oficial e depuração de integrações OpenAI.

Quando acionar:
- Construção ou revisão de aplicativos baseados em OpenAI.
- Uso de Agents SDK, ChatGPT Apps, API Platform ou chaves locais.
- Erros de autenticação, chamadas de API, modelos, ferramentas ou respostas inesperadas.
- Necessidade de confirmar comportamento atual de produtos OpenAI em documentação oficial.

Como agir:
- Verificar fontes oficiais quando a informação puder ter mudado.
- Separar configuração, código, credenciais e comportamento observado.
- Propor passos mínimos, testáveis e reversíveis.

Limite: não transformar a orientação em documentação genérica longa quando a demanda pedir apenas uma ação concreta.
