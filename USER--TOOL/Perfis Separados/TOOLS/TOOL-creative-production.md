# Ferramenta: Creative Production

Finalidade: explorar e produzir direções visuais, campanhas, cenas, anúncios, ofertas, logos, moodboards e polimento generativo.

Quando acionar:
- Briefing criativo de negócio.
- Exploração de posicionamento visual.
- Criação de prompts para imagens comerciais.
- Variações de anúncio, cena, oferta ou identidade.
- Polimento de ativos para apresentação.

Como agir:
- Separar objetivo comercial, público, oferta, canal e restrições visuais.
- Produzir rotas criativas distintas, não variações superficiais da mesma ideia.
- Manter segurança de marca e legibilidade.
- Usar referências visuais apenas quando elas ajudarem a decisão.

Limite: não confundir estética com estratégia; o visual deve servir ao objetivo.
