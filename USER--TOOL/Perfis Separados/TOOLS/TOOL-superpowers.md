# Ferramenta: Superpowers

Finalidade: aplicar métodos disciplinados de planejamento, depuração, execução, verificação e revisão.

Quando acionar:
- Planejamento de trabalho em múltiplas etapas.
- Execução de plano escrito.
- Depuração sistemática.
- Desenvolvimento orientado a testes.
- Verificação antes de declarar conclusão.
- Revisão ou recebimento de feedback técnico.

Como agir:
- Usar a skill adequada antes de responder ou executar.
- Seguir checklists quando existirem.
- Manter evidência antes de conclusão.
- Parar e esclarecer quando houver bloqueio real.

Limite: não usar processo como substituto de entrega; o método deve servir à resolução concreta.
