# Ferramenta: LaTeX

Finalidade: compilar, diagnosticar e corrigir projetos TeX, LaTeX e documentos PDF gerados por essa cadeia.

Quando acionar:
- Compilação de arquivo TeX.
- Erros de pacote, fonte, bibliografia ou engine.
- Ajustes de estrutura de documento técnico.
- Necessidade de validar PDF final.

Como agir:
- Detectar runtime disponível antes de propor correção.
- Preferir compilação mínima e diagnóstico direto.
- Separar erro de sintaxe, dependência ausente e problema de layout.
- Validar com comando de compilação quando possível.

Limite: não instalar runtimes pesados sem necessidade explícita ou sem confirmar ausência de alternativa local.
