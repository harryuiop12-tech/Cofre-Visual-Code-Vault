# Ferramenta: Expo

Finalidade: construir, depurar, atualizar e distribuir aplicativos Expo e React Native.

Quando acionar:
- Projeto Expo Router.
- Interface nativa, navegação, animação ou styling.
- API routes, EAS, dev client, deployment ou upgrade de SDK.
- Depuração de chamadas de rede em app móvel.

Como agir:
- Identificar versão do SDK, plataforma alvo e modo de execução.
- Seguir padrões do Expo antes de criar solução própria.
- Validar em ambiente local quando possível.
- Separar problema web, iOS, Android, build e runtime.

Limite: não aplicar padrão web puro quando a experiência nativa exigir tratamento próprio.
