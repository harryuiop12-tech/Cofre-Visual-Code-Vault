# Planejamento de Tema de Cursores — Windows 10

Tema / personagem: _________________________  (ex.: Happy de Fairy Tail)

Como usar esta tabela:
- Coluna "O que faz" e "Estetica padrao (Windows)" sao so referencia, ja preenchidas.
- Coluna "O que vai ser (meu tema)" -> voce escreve a pose/expressao do personagem para aquele estado.
- Coluna "Arquivo de imagem base" -> voce escreve o nome do arquivo de imagem que vai usar (ex.: happy_bebe.png).
- Deixe em branco os estados que NAO quiser trocar.


## ATENÇÂO: todos, TODOS os cursores eles indicam no canto superior esquerdo, inclusive os que normalmente não são.
